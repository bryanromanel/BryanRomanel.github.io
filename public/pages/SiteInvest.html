Site Invest Now.

Abaixo encontra-se link para download do projeto "Invest now" construído com as linguagens HTML, CSS, e, Javascript.

Além disso, foi incluído um projeto de terceiros, que utiliza o Web GL para realizar uma animação de fluidos.

Este projeto foi realizado para a disciplina de Experiência Criativa da Pontifícia Universidade Católica do Paraná no primeiro semestre do ano de 2021.

Foi desenvolvido em conjunto com Beatriz Zardo, Gustavo Viana do Nascimento, e, Vitor Wunsch.

Para executar este site é necessário a instalação do programa XAMPP e a extração dos arquivos na pasta htdocs dos arquivos do programa.

Clique aqui para ser direcionado à página do XAMPP.
